# Formatting Guidelines

## Text Emphasis
- **Bold**: Important keywords only (2-3 per slide max)
- *Italic*: Supplementary info, quotes
- `Code`: Technical terms, specific values

## Lists
- Bullet lists: 3-7 items maximum
- Nesting: 2 levels maximum
- Keep items short (phrases, not sentences)

```markdown
- Main point
  - Sub point (max 2 levels)
```

## Numbered Lists
```markdown
1. First step
2. Second step
3. Third step
```

## Images
```markdown
![Alt text](./images/filename.png)
```

## Code Blocks
````markdown
```python
print("Hello")
```
````

## Tables
```markdown
| Header 1 | Header 2 |
| :------- | -------: |
| Left     | Right    |
```
